# Odin Recipes

This project is part of **The Odin Project** curriculum. The goal of this project is to build a simple website showcasing various recipes.

## Purpose

- Practice basic HTML skills.
- Understand how to structure a simple webpage.
- Gain hands-on experience with Git version control and repository management.

## Skills Demonstrated

- Writing semantic HTML.
- Using Git for version control.
- Setting up and managing a GitHub repository.
